(async) loadPosts(searchPostsopt) → {Promise.<void>}

Loads and displays posts in the home feed. If search results are provided, it displays them; otherwise, it fetches all posts. Each post is rendered with the author's details, title, body, and any associated media. Additionally, the function conditionally displays a follow/unfollow button based on whether the post author is the current user, and provides options to edit or delete the post if the current user is the author.
Parameters:
Name Type Attributes Default Description
searchPosts Array.<Object> | null <optional>
 null Optional array of post objects to render instead of fetching.

Handles the submission of the "Create Post" form. Validates the user input, displays notifications on errors, and sends the post data to the API. On success, resets the form, closes the modal, and reloads the post feed.
